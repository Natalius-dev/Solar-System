Solar System

Open model Open docs

This visualization was made in a few days using only Javascript, HTML, and CSS. Three.js and dat.gui were used in the 3D visualization made by Natalius Gabriel, while the home page was made in HTML and CSS with MD Bootstrap by Benedict Halim. The home page contains information on all the planets along with the Sun and our solar system. The 3D visualization plots the major planet's and moon's orbits using data provided by NASA (though they are not to scale for better viewing purposes.) All of the textures used are accurate to some degree though some of them have had artistic liberties taken because we have not been able to map the entire body yet.

3D Javascript Code

Made using Three.js and dat.gui by Natalius Gabriel.

Using Three.js, we take the orbital data of all the major planets and moons in the solar system and plot them by making an ellipse using the provided semi-major axis and calculating the semi-minor axis using the orbit's eccentricity. Then we put all the points in the ellipse inside an array that we can continuously loop through to move the planets and moons. Dat.gui allows us to set different variables straight from the browser so we can interact with the visualization easier.
